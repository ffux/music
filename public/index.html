About

Music Downloader

A self-hosted Apple Music downloader and library browser. Search for music, download lossless ALAC files, and play them straight from the browser.

Stack

Backend: Node.js + Express, better-sqlite3, WebSockets
Downloader: gamdl (Python) — Apple Music ALAC via cookies auth
Audio: FFmpeg for processing
Process: Supervisor (Node + Python worker)
Infra: Docker → Caddy → Cloudflare Tunnel (vibedeploy)

How it works

Paste an Apple Music URL or search for music. The download is queued and processed by the Python worker using gamdl with your stored cookies. Files are saved as ALAC .m4a and served from the Library tab.
